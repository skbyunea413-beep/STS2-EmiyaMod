# 03. Common 카드 풀

> **상태**: 🔲 미완 — 카드 목록 초안 단계

---

## 설계 방향

- 마력 코스트 없음 (Common은 에너지만).
- 기본 강화/방어/투영 테마 확립.
- 투영 카드([Traced]) 3~4종 포함 — UBW 연동 렐릭 의미 있게 쌓이도록.

## 미결 사항

- [ ] Kanshou/Bakuya 페어 메커니즘 — 두 카드가 짝으로 작동하는 방식 확정
- [ ] Projection 카드가 생성하는 "투영 카드"의 구체 대상 결정
- [ ] Double Cast의 2회 시전이 마력 코스트도 2번 내는지 여부
- [ ] Common 총 카드 수 목표: Attack 5종, Skill 5종, Power 2종 = 12종
