# 01. 이중 자원 시스템 설계

---

## 두 자원의 역할 분리

| 자원 | 회복 시점 | 용도 | 성격 |
|------|-----------|------|------|
| **에너지** | 매 턴 3 회복 | 일반 카드 모두 | 턴당 예산 |
| **마력(★)** | 전투 시작 시만 지급 | 투영 Noble Phantasm | 전투 단위 예산 |

에너지는 STS2 기본 자원. 마력은 덱 전체를 관통하는 **희소 자원**.  
에너지가 다 떨어지면 턴이 제약되고, 마력이 다 떨어지면 강수를 못 둔다.

---

## 마력 수급 방법

| 방법 | 상세 |
|------|------|
| **Starter 렐릭** (Magic Circuit) | 전투 시작 시 ★3 지급 (기본) |
| **카드** (Mana Charge) | 에너지 1 소비 → ★2 획득 |
| **렐릭** (Second Circuit) | 전투 시작 ★ +2 추가 |
| **파워** (Mana Surge) | 매 3턴마다 ★1 획득 |

**기본 전투 예산**: ★3 (Starter 렐릭만 있을 때)  
최대 확장 시 ★8~10 정도가 한 전투에서 쓸 수 있는 마력.

---

## 카드 코스트 설계 원칙

```
일반 카드:          에너지만 소비     (예: 1에너지)
투영 카드 (약):     에너지 + ★1      (예: 1에너지 + ★1)
투영 카드 (강):     에너지 + ★2      (예: 1에너지 + ★2)
투영 카드 (NP):    낮은 에너지 + ★2~3 (예: 0에너지 + ★2, 1에너지 + ★3)
```

**의도**: 마력이 에너지를 대체하는 것이 아니라 **추가** 소비.  
마력을 아낄수록 평범하게 플레이하고, 마력을 쓸수록 터진다.

---

## 마력 코스트 예시 (카드별)

| 카드 | 에너지 | 마력 | 효과 요약 |
|------|--------|------|-----------|
| Reinforce | 1 | ★0 | 6 피해 + 강도 버프 |
| Trace — Sword | 1 | ★0 | 8 피해, Exhaust [Traced] |
| Kanshou & Bakuya | 2 | ★0 | 6+6 피해 [Traced] |
| Caladbolg II | 1 | ★1 | 20 피해 + Vulnerable [Traced] |
| Hrunting | 1 | ★1 | 12 피해 × 2 [Traced] |
| Gáe Bolg (복사) | 0 | ★2 | 치명타 효과 포함 30 피해 [Traced] |
| Caliburn | 1 | ★2 | 45 피해, 전투 1회 [Traced] |
| Rule Breaker | 1 | ★2 | 10 피해 + 적 파워 전부 제거 [Traced] |
| UBW (파워) | 3 | ★2 | 파워: 매 턴 투영 카드 뽑기 + 스케일 |

---

## 균형 기준점

- ★1 투영 카드: 같은 에너지 일반 카드의 약 **2배** 효율
- ★2 투영 카드: 같은 에너지 일반 카드의 약 **3~4배** 효율
- 마력이 전혀 없을 때도 에너지 카드만으로 클리어 가능해야 함 (Hard mode)
- 마력을 풀로 활용하면 압도적인 폭발력 (Reward)

---

## 구현 노트

```csharp
// 마력 코스트는 CanonicalStarCost로 구현
public override int CanonicalStarCost => 1;  // ★1 소비

// 에너지 코스트는 CanonicalEnergyCost
protected override int CanonicalEnergyCost => 1;

// 카드 설명문 예시: "1 에너지, ★1 | 20 피해. 대상에게 취약 부여."
// Stars 아이콘: {Stars:starIcons()} 포맷터 사용
```
